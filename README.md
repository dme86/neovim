## installation

git clone with submodules:

    git clone --recurse-submodules git@github.com:dme86/neovim.git
    
Run `yarn install` inside `~/.config/nvim/pack/plugins/start/coc.nvim` and type `:CocInstall coc-go` inside neovim.
